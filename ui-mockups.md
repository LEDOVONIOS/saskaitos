## 📱 Mobile Navigation Pattern

```javascript
// Mobile menu toggle
Alpine.data('mobileMenu', () => ({
    open: false,
    
    toggle() {
        this.open = !this.open;
        document.body.style.overflow = this.open ? 'hidden' : '';
    },
    
    close() {
        this.open = false;
        document.body.style.overflow = '';
    }
}));
```

```html
<!-- Mobile Header -->
<header class="mobile-header" x-data="mobileMenu">
    <button @click="toggle()" class="menu-toggle">
        <svg class="icon" viewBox="0 0 24 24">
            <path x-show="!open" d="M3 12h18M3 6h18M3 18h18"/>
            <path x-show="open" d="M6 18L18 6M6 6l12 12"/>
        </svg>
    </button>
    
    <h1 class="mobile-title">Sąskaitos Pro</h1>
    
    <!-- Mobile Sidebar Overlay -->
    <div x-show="open" 
         @click="close()" 
         class="sidebar-overlay"
         x-transition:enter="transition ease-out duration-300"
         x-transition:enter-start="opacity-0"
         x-transition:enter-end="opacity-100">
    </div>
</header>
```

## 🎨 Color Usage Guidelines

### Primary Actions
- Create Invoice: Primary Blue (#2563eb)
- Save Draft: Secondary Gray (#64748b)
- Delete: Danger Red (#ef4444)

### Status Colors
- Paid/Success: Green (#10b981)
- Pending/Warning: Amber (#f59e0b)
- Overdue/Error: Red (#ef4444)
- Draft: Gray (#64748b)

### Background Hierarchy
- Main Background: #f8fafc
- Card Background: #ffffff
- Hover States: #f1f5f9
- Selected States: #e0e7ff
